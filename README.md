
===========

Apptireman with CSS Framework based on material design

### Current Version : v2.55.0

- Descomprimir archivo en XAMP Server en directorio wssp_app
- Configurar la DB con el archivo sql adjunto.


## Compiling Files
- `npm install`
- `grunt monitor`, this will compile .scss, .js., .jade files.
